# Landslide Segmentation with Graph Neural Networks

Source-only research repository for landslide segmentation with AMMG-UNet and Graph U-Net. Downloaded datasets, checkpoints, generated results, archives, and local environment files are intentionally excluded from version control.

The implementations are being kept as separate experiments because they use different graph representations:

- `ammg_unet/`: pixel segmentation based on the AMMG-UNet paper architecture.
- `graph_unet/`: Graph U-Net experiments, including the Colab notebook supplied with this project.

See the component READMEs for their current setup instructions. A consolidated reproducibility guide and research citations are added in the documentation commit.
